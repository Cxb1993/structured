00021      Extract, from the result of program delauny, the facets of the surface of the scene,\n\
00022      and write the result to a ply file in binary format\n\
00023 \n\
00024      The program first eliminates empty tretrahedrons\n\

00027      <input_cgal> : data generated by program **delaunay**.\n\
00028      <out_ply> : .ply output file.\n\
00029      -e : extraction mode :\n\
00030           - 0 = use removed tretrahedrons\n\
00031           - 1 = use kept tretrahedrons\n\
00032           - 2 = extract all facets of removed tretrahedrons (for test purpose)\n\
00033           - 3 = extract all facets of kept tetrahedrons (for test purpose)\n\
00034      -i : remove tetrahedrons with at least <min_intersect> rays going through them.\n\
00035      -iw : use a number of intersections larger than <min_intersect> for \"large\" tetrahedrons\n\
00036      -mxf : use maxflow algorithm, with a threshold of <nb-intersections>, to remove tetras\n\
00037      -a : remove facets that do not satisfy a constraint on normal angles.\n\
00038          + if <normals-angle> > 0, rm facets where the angle between the facet normal and a vertex normal is greater than angle\n\
00039          + if <normals-angle> < 0, rm facets where the angle between the facet normal and the sum of vertex normals is greater than |angle|.\n\
00040            Angle value from 0 to 180; default value 180 means \"no angle constraint\".\n\
00041      -ct : remove tetrahedrons that have 3 facets on the surface (rough patches).\n\
00042      -sm : try to smooth the surface. Replace each point p by <lambda> * p + (1 - <lambda>) * neighbours_mean; repeat <nb_iter> times.\n\
00043            if <lambda> <= 0., use default value of 0.8; if <nb_iter> < 0, do a weighted mean based on normal angles.\n\

00048   exit(1);
